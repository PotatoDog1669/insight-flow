Prefix: `/api/v1/tasks`

## Purpose

Inspect or trigger lower-level collection tasks.

## Main Endpoints

- `GET /api/v1/tasks`
- `POST /api/v1/tasks/trigger`
- `GET /api/v1/tasks/{task_id}`

## Common Scenarios

- Debug task state transitions
- Trigger a direct collection task
