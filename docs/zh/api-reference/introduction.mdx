Insight Flow 的 API 以 `/api/v1/` 为统一前缀，面向前端管理界面和自动化调用。

## 基本约定

- 资源接口集中在 `/api/v1/`
- OpenAPI 文档位于 `http://localhost:8000/docs`
- 健康检查位于 `/health`
- 前端调用统一收敛在 `frontend/src/lib/api.ts`

## 资源分组

- Sources
- Articles
- Tasks
- Reports
- Monitors
- Providers
- Destinations
- Feed
- Users
