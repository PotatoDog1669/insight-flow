# 快速开始

## 前置条件

- Python 3.12+
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 16 & Redis 7

## 启动步骤

1. 克隆仓库并配置环境变量：

```bash
git clone <repo-url> && cd insight-flow
cp .env.example .env
# 编辑 .env 填入你的 API Key
```

2. 启动基础服务：

```bash
docker compose up -d postgres redis
```

3. 启动后端：

```bash
cd backend
pip install -e ".[dev]"
alembic upgrade head
uvicorn app.main:app --reload --port 8000
```

4. 启动前端：

```bash
cd frontend
npm install
npm run dev
```

5. 访问应用：
   - 前端：http://localhost:3018
   - API 文档：http://localhost:8000/docs
